# IAA 计划

IAA 计划，即 INPUT ANYTHING ANYWHERE 计划。

该计划旨在为任何支持自定义词库的输入法和任何支持插件机制的软件提供
[ConL-IME] 项目中各语言的码表。从而实现在任何地方能输入任何文字和符号。

[ConL-IME]: https://github.com/aj-ash/ConL-IME
